应用名称：JaySay课程表
版本：v1.4
大小：51MB
系统要求：Android 7.0+

=== 应用简介 ===

JaySay课程表是一款专为大学生设计的课表管理App，支持从教务系统导出的Excel一键导入课程，自动生成美观的周视图课表。

=== 核心功能 ===

多课表管理 — 支持创建多个课表（如大一/大二/选修课），随时切换，互不干扰

Excel批量导入 — 支持手机文件管理器直接选择教务系统导出的.xlsx文件，一键导入全部课程，自动识别周次、节次、教师、教室

灵活周次设置 — 支持范围选择和不连续周自由选择，完美适配"单双周""隔周上课"等复杂排课场景，支持单双周自动识别

自定义节次时间 — 12节课时间均可自定义，适配不同学校的作息时间表

智能化体验 — 设置开学日期后自动计算当前周，打开App直接显示本周课表，无需手动翻找。左右滑动快速切换周次，固定表头始终显示日期和星期

视觉优化 — 课程卡片彩色编码，课程名/教师/教室完整显示不截断，支持深色模式、自定义背景图片、跟随系统主题，界面清爽高级

数据安全 — 本地SQLite存储，无需网络权限，数据完全保存在手机本地

=== 使用说明 ===

1. 从学校教务系统导出课表Excel文件
2. 点击App右上角"导入"按钮，在手机文件管理器中找到该文件
3. App自动解析并显示课程预览，确认后导入
4. 在设置中可调整学期开始日期、节课时间、外观主题
